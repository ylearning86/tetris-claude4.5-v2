# テトリスゲーム 🎮

HTML/CSS/JavaScriptで作成したクラシックなテトリスゲームです。

## 🎯 プロジェクト概要

このプロジェクトは、プログラミング学習を目的として、GitHub CopilotとVS Codeを使用して開発されたブラウザベースのテトリスゲームです。

## ✨ 機能

- 🎲 7種類のテトロミノ（I, O, T, S, Z, J, L）
- 🔄 ブロックの回転と移動
- 🎯 ライン消去とスコアリング
- 📊 レベルシステム（10ライン毎にレベルアップ）
- 👀 次のブロック表示
- 🔊 効果音（Web Audio API使用）
- 📱 レスポンシブデザイン

## 🎮 操作方法

| キー | 動作 |
|------|------|
| `←` / `→` | 左右移動 |
| `↓` | 高速落下 |
| `↑` / `スペース` | 回転 |
| `P` | 一時停止/再開 |

## 🎵 効果音

- **移動音** - ブロックを左右に移動したとき
- **回転音** - ブロックを回転させたとき
- **着地音** - ブロックが底に着いたとき
- **ライン消去音** - ラインが消えたとき
- **レベルアップ音** - レベルが上がったとき
- **ゲームオーバー音** - ゲームが終了したとき

## 🚀 使い方

### ローカルで実行

1. このリポジトリをクローン
```bash
git clone https://github.com/あなたのユーザー名/tetris-claude4.5-v2.git
```

2. `index.html`をブラウザで開く
```bash
cd tetris-claude4.5-v2
start index.html  # Windows
# または
open index.html   # Mac
```

### GitHub Pagesで公開

1. GitHubリポジトリの Settings に移動
2. 左サイドバーの「Pages」を選択
3. Source セクションで「Deploy from a branch」を選択
4. Branch を「main」、フォルダを「/ (root)」に設定
5. 「Save」をクリック
6. 数分後、`https://あなたのユーザー名.github.io/tetris-claude4.5-v2/`でアクセス可能

## 🔄 開発フロー

1. **コーディング (VS Code + GitHub Copilot)**
   - GitHub CopilotがAIアシスタントとしてコード生成をサポート
   - リアルタイムでコード補完と提案を提供

2. **ローカルテスト**
   - ブラウザで`index.html`を開いて動作確認
   - 開発者ツールでデバッグ

3. **バージョン管理 (Git)**
   ```bash
   git add .
   git commit -m "機能追加: 効果音の実装"
   git push origin main
   ```

4. **リモートリポジトリ (GitHub)**
   - コードの安全な保存
   - 変更履歴の追跡
   - 他の開発者とのコラボレーション

5. **デプロイ (GitHub Pages)**
   - mainブランチへのpush時に自動デプロイ
   - 数分以内に公開URLで利用可能
